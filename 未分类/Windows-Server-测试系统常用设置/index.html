Windows Server 测试系统常用设置

发表于 2026-05-18 分类于 未分类

禁用 IPv6

Win + R 打开注册表 regedit

访问：计算机\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Tcpip6\Parameters

名称为：DisabledComponents

双击修改，设置值为：ffffffff

重启后生效
